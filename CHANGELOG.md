## [1.0.0] - 2025-XX-XX

### Added
- Initial release
- Support for Spanish and Turkish languages
- 7 lesson categories per language
- Interactive dialogue-based lessons
- Progress tracking per language and category
- Dark mode support
- Text-to-Speech pronunciation (premium feature)
- Settings modal with language and category selection
- Swipe navigation between lessons
- Auto-close menu after selection
- Multi-language architecture
- Language-specific audio storage in Google Cloud

### Features
- 🆓 Free: Complete access to all lessons
- 💎 Premium: Native pronunciation with Google Cloud TTS

### Technical
- Built with React Native and Expo
- FastAPI backend
- Google Cloud Text-to-Speech integration
- Google Cloud Storage for audio caching
- AsyncStorage for local progress persistence
- GPL-3.0 license

## Version Format

- **Major.Minor.Patch** (e.g., 1.0.0)
- **Major**: Breaking changes
- **Minor**: New features (backwards compatible)
- **Patch**: Bug fixes (backwards compatible)

## Categories

- **Added**: New features
- **Changed**: Changes in existing functionality
- **Deprecated**: Soon-to-be removed features
- **Removed**: Removed features
- **Fixed**: Bug fixes
- **Security**: Security improvements
